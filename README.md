## üé® Design Philosophy

### **Design Thinking Integration**

- **3P Culture**: Purpose, Process, People integrated into every interaction.
- **5 Pillars**: Learning, Upskilling, Innovation, Networking, and Character Building.

- **Modern UI/UX**: Built with premium TailwindCSS styling and smooth animations.

- **Interactive Elements**: Floating WhatsApp and Instagram integration for direct communication.
- **SEO Ready**: Structured data, meta descriptions, and semantic HTML implemented.

## üìù Getting Started

### **Local Development**
```bash
# Install dependencies
npm install

# Run development server
npm run dev

# Build for production
npm run build
```

### **Deployment**
The project is ready for deployment on **Cloudflare Pages** or any Node.js environment.
```bash
# Build the project
npm run build

# Deploy using Wrangler (for Cloudflare)
npx wrangler pages deploy dist --project-name dr-sns-education
```
